##  About the Project

**Namma UI Project** is a lightweight frontend project built using **pure HTML and JavaScript**.

It focuses on:
- Building UI from scratch  
- DOM manipulation  
- Responsive design  
- Clean code  

---

##  Features

- ⚡ Fast and lightweight  
- 📱 Responsive design  
- 🎨 UI sections:
  - Kadak Chai ☕  
  - Adrak Chai 🔥  
  - Bangalore Rain 🌧️  
  - Garden 🌿  
- 🔢 Pagination (1 / 2 switching)
